# Castellary MVP — Asset Reference

This document maps all images, backdrops, character illustrations, and audio tracks used across Castellary.

---

## 🎨 Layout Frames & Graphics

* **`/logo/crest.png`**: The main emblem logo of Castellary, optimized using Next.js Image component with priority loading.
* **`/logo/shield.png`**: The medieval shield card container, used in Slide 2 and Slide 10 steps.

## 🔊 Sound Effects & Ambient Audio

* **`/audio/hover.wav`**: Played when mouse hovers over menu options and CTA buttons.
* **`/audio/click.wav`**: Played when clicking triggers and navigation links.
* **`/audio/victory.wav`**: Played when a user overrides themes or completes quests.
* **`/audio/damage.wav`**: Played when dealing strike damage to the raid boss.
